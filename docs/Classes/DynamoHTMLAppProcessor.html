DynamoHTMLAppProcessor

Generated class to ease creation of text for common HTML tags. tr( td( text ) ) is converted into text. By convention if the contents are nil the tag is not closed. A optional set of attributes acn be passed as a dictionary: a( [href=htp://google.com], link text ) becomes link text.
